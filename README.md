# Projeto 1 IaC para processamento de dados com AWS EMR e Apache Flink
O objetivo principal deste projeto é automatizar a configuração e o gerenciamento da infraestrutura necessária para executar pipelines de processamento de dados em lote e streaming, capazes de lidar com grandes volumes de dados com baixa latência. Automatizei o provisionamento da Infraestrutura de engenharia de dados através de IaC com Terraform no ambiente AWS utilizando o serviço EMR com cluster e o pacote de aplicativo Apache Flink.
